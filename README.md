#  MHiT DB 프로젝트

야구 경기와 셔틀 예약을 한 번에!

&nbsp;

---

## 📌 프로젝트 개요  

- **MHiT** : 사용자가 원하는 야구 경기를 예매할 때 근방에서 해당 경기장까지 운행하는 셔틀 버스까지 예약할 수 있는 DB 중심의 프로젝트입니다.

- 실제 서비스 운영에 적합한 **ERD**를 설계하고, **SQL 쿼리와 프로시저가 요구사항대로 정확하게 실행되는지 테스트**하는 데 중점을 두었습니다.
